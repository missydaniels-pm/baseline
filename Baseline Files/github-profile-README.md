## What I'm building

### [Baseline](https://github.com/missydaniels/baseline) *(in progress)*

A tool for people managing chronic health conditions who want real data on what's actually working.

Most health apps just log symptoms. Baseline gives you the structure to run real experiments — establish a baseline, introduce one variable, track the results, and build an evidence base for your own health decisions. The scientific method, applied to personal health.

**Built with:** Python · Flask · SQLite · Anthropic API · Jinja2

**Why this:** I know this problem firsthand. And building something real — with actual users — felt like the right way to learn.
